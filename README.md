# Basic Programming Concepts
C# Learning Repository

This repository contains a collection of simple C# programs created while learning the fundamentals of the language. The code focuses on illustrating the essential concepts and syntax that form the foundation of C# development.

Overview

Each file demonstrates a specific aspect of C# basics, from console input and output to control structures and object-oriented principles. The repository serves as a compact reference for understanding how core language elements work together in simple applications.

Topics Covered

Basic Syntax and Data Types – variables, constants, and primitive types

Input and Output – console operations using Console.ReadLine() and Console.WriteLine()

Control Flow – conditional statements and looping constructs

Methods – declaration, parameters, return values, and overloading

Arrays and Strings – basic data handling and iteration

Exception Handling – using try, catch, and finally

Basic Object-Oriented Concepts – classes, objects, and constructors

Purpose

The purpose of this repository is to consolidate the core building blocks of C# in a clear and practical manner, providing a quick reference for anyone beginning their journey with the language.
